
# Conservative PID

Welcome to the documentation for the Conservative PID library.

## Getting Started

This library provides tools for causal inference using:
- **Symbolic Language**: Define variables, interventions, and counterfactual terms.
- **Canonical Basis**: Represent causal models as canonical configurations (worlds).
- **Solver**: Compute bounds on causal queries using Linear Programming.

## Installation

```bash
uv sync
```
