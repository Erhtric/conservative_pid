
# Conservative CPID

Welcome to the documentation for the *Conservative Partial IDentification of causal effects* (CPID) library.

## Getting Started

This library provides tools for causal inference using:
    - **Symbolic Language**: Define variables, interventions, and counterfactual terms.
    - **Canonical Basis**: Represent causal models as canonical configurations (worlds).
    - **Solver**: Compute bounds on causal queries using Linear Programming.
    - **Infernece**: Entry-point interface.

## Installation
I would suggest to install the excellent package manager `uv` by Astral. For a speedy setup please visit their `Getting started` webpage: https://docs.astral.sh/uv/getting-started/.

```bash
uv sync
```
